# Check_Unauth
常见未授权检测合集，ftp、Memcached、mongodb、zookeeper、Redis、elasticsearch未授权检测

usage: python3 Check_Unauth.py 127.0.0.1

1.mongodb默认端口和27017

2.zookeeper默认端口2181

3.Redis默认端口6379

4.elasticsearch未授权检测脚本默认端口9200

5.ftp、Memcached
